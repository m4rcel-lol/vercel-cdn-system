# Images Directory

This directory contains image files that can be accessed via the CDN.

## Supported Formats
- JPG/JPEG (`.jpg`, `.jpeg`)
- PNG (`.png`)
- WebP (`.webp`)
- SVG (`.svg`)
- ICO (`.ico`)

## Usage
Files in this directory will be available at:
```
https://your-domain.vercel.app/files/images/your-image.jpg
```
