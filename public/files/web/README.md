# Web Directory

This directory contains web-related files that can be accessed via the CDN.

## Supported Formats
- HTML (`.html`)
- CSS (`.css`)
- JavaScript (`.js`)
- JSON (`.json`)
- XML (`.xml`)

## Usage
Files in this directory will be available at:
```
https://your-domain.vercel.app/files/web/your-script.js
```
